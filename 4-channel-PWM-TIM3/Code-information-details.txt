- PWM with timers project

- Four channels PWM (PA6 CH1, PA7 CH2, PB0 CH3, PB1 CH4)

- To see different effects, I do: 
    __HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_1, duty);
    __HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_2, 1000 - duty);
    __HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_3, duty / 2);
    __HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_4, 500 - (duty / 2));

- Watchdog timer IWDG. For a 10 seconds delay, watchdog values are:
Prescaler: 256 (the highest it can go, Prescalers = {4, 8, 16, 32, 64, 128, 256})
Reload: 1249 
- When I press the B1 push button, the code gets stuck inside a while(1), 
which triggers (after those 10 seconds) the watchdog.

- Timer TIM2 to blink an LED at 200ms LOW, 200ms HIGH. Values below will
produce a 10ms interrupt, which has to be counted up to 20 to produce 200ms:
  Timer interrupt: TIM2
  TIM2 values for 10ms interrupt:	
  htim2.Instance = TIM2;
  htim2.Init.Prescaler = 84-1;
  htim2.Init.CounterMode = TIM_COUNTERMODE_UP;
  htim2.Init.Period = 10000-1;
  htim2.Init.ClockDivision = TIM_CLOCKDIVISION_DIV1;
  htim2.Init.AutoReloadPreload = TIM_AUTORELOAD_PRELOAD_DISABLE;
