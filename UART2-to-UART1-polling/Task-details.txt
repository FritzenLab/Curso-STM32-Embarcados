- USART example:

- Polling, when button B1 (PC13) is pressed, an incremental number is sent
from TX (PA9) to RX (PA10) of UART1, at 9600 bps. 

-LED2 (PA5) blinking from a 10ms TIM2 interrupt (that counts to 20,so 200ms).

- Usart2 (PA2 TX, PA3 RX) transmits to Putty (at 115200 bps) the counter value


- Timer TIM2 to blink an LED at 200ms LOW, 200ms HIGH. Values below will
produce a 10ms interrupt, which has to be counted up to 20 to produce 200ms:
  Timer interrupt: TIM2
  TIM2 values for 10ms interrupt:	
  htim2.Instance = TIM2;
  htim2.Init.Prescaler = 84-1;
  htim2.Init.CounterMode = TIM_COUNTERMODE_UP;
  htim2.Init.Period = 10000-1;
  htim2.Init.ClockDivision = TIM_CLOCKDIVISION_DIV1;
  htim2.Init.AutoReloadPreload = TIM_AUTORELOAD_PRELOAD_DISABLE;

- There is protection to only send information to Putty via USART2 when 
the whole package of bytes have arrived.
